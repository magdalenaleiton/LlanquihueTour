# Sistema Agencia de Turismo Llanquihue Tour

## Descripción
Proyecto desarrollado en Java utilizando Programación Orientada a Objetos para representar personas relacionadas con una agencia de turismo.

## Estructura

### Paquete model
- Persona
- Direccion
- GuiaTuristico

### Paquete app
- Main

## Conceptos aplicados
- Encapsulamiento
- Herencia
- Composición
- Constructores
- Getters y Setters
- Método toString()

## Ejecución
1. Abrir el proyecto en IntelliJ IDEA.
2. Ejecutar la clase Main.
3. Revisar la salida en consola.
